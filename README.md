# JBuy (Spring Boot + Angular)
An e-commerce website using Spring Boot and Angular
(in development)
## Demo
https://jbuy-shop.herokuapp.com

## Features
- Rest API
- Product search 
- Full text search
- Faceted search
- Search text autocomplete
- Shopping cart, checkout, review
- OpenAPI

## Technology Stacks
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- Hibernate Search
- Lucene
- Hazelcast
- Springdoc
- Rapidoc

- Angular
- Angular Material
- Elf

## Requirement:
```
Java 11 or above
Node 16 or above
npm 8.5 or above
Angular/cli 13 or above
```

## Run dev mode
- Spring
```
---From root folder
$ cd api
$ ./mvnw clean install
$ ./mvnw spring-boot:run
```

- Angular
```
---From root folder
$ cd webapp
$ npm install
$ ng serve -o
```
